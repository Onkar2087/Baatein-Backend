Baatein Backend — Real-Time Chat API

This is the backend service for a real-time chat application built with Node.js and Express. It handles user authentication, real-time messaging, and media uploads while ensuring security and performance.

The system uses JWT for authentication, Socket.IO for instant communication, and Cloudinary for efficient image storage. MongoDB is used as the database with optimized queries and indexing to support fast data retrieval.

✨ Features
🔐 JWT Authentication (secure login/signup)
💬 Real-time messaging with Socket.IO
🖼️ Image upload using Multer + Cloudinary
📦 RESTful APIs for users, conversations, and messages
⚡ MongoDB indexing for performance
🛡️ Rate limiting for API protection
